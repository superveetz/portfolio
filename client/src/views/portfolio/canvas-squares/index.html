Canvas Squares

I'm a big fan of using Canvases in web pages. They are extremely flexible in the ways that they can be used and they add unique effects that can not be achieved by standard web technologies.

This experiment allows a user to place a large number of square particles anywhere on a canvas and set checkpoints for which each square will move towards. By using the A* path finding algorithm, I was able to ensure that no collisions or overlaps occur while each square moves around the screen.

Technologies:

  • Canvas
  • AngularJS
  • Bootstrap
